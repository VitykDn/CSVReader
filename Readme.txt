CSVReader project
CSVReader is an Asp.Net Core MVC WebApp that allows allows you to convert data from csv format files into a web table. The downloaded data is stored in a MySQL database.

Technologies used:
- .NET 6.0
- MySQL 
- ASP.NET CORE 6
- jQuery Ajax
- CsvHelper

 Completed technical tasks:
- Creation and filling of a web table with defined fields using csv files.
- Possibility to search for data in any fields in the table.
- Sort function by clicking on the column header.
- Inline editing for almost any column in the table.
- possibility to remove a record from the database.
- Validation and logging.

File "Test7.csv"in the root folder has data for testing.

Encountered Problems
 Inline editing for boolean is currently buggy and disabled.
